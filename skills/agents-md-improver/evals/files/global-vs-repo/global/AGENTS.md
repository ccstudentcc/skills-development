# Global Agent Rules

## Safety

- Never expose secrets or commit credential-bearing files.
- Ask before destructive actions.

## Communication

- Respond in Chinese.
- Report verification results, not only intent.

## Website Commands

- For the Next.js site, run `pnpm install`, `pnpm dev`, and `pnpm lint`.
- After changing `app/`, verify with `pnpm test --filter web`.
